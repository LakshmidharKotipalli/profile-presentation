Project 01 · Public

DocMe

Privacy-First macOS Activity Tracker with AI Daily Summaries
View on GitHub Released as DMG
Problem

Knowledge workers have no coherent record of their workday. Existing tools (RescueTime, Toggl) send data to cloud servers — a serious privacy risk for anyone handling proprietary code or sensitive documents.

Solution

Native macOS Electron app. Python agent polls every 1.5s — active windows, URLs via browser extension, file changes. Local SQLite. llama.cpp + gemma generates natural-language daily summaries on demand. Zero config.

Key Design Decisions
→ Zero cloud — everything stays on disk
→ Bundled Python runtime — no install required
→ Chrome/Firefox extension for URL capture
→ Quick Notes capture current app context
→ SQLite default — no database server needed

Project 02 · Public

GeoCanvas

Location-Aware AI Image Generation
View on GitHub
Problem

Standard text-to-image tools have no sense of place. Writing a location-accurate prompt requires knowing the exact place name — the prompt is the hardest part, not the generation itself.

Solution

Click any point on Earth on an interactive Leaflet map → Nominatim reverse geocodes coordinates → auto-fills an editable prompt → Stable Diffusion XL generates a 1024×1024 image. 4 presets included.

Pipeline
1 Map click → lat/lng state
2 Nominatim reverse geocode → place name
3 Auto-filled, editable prompt
4 POST /api/generate → SDXL pipeline
5 PNG returned as base64 → display + download
Stack

Architecture

Recruiter Agent

AI-Powered Candidate Sourcing & Resume Matching
Private Repo
Problem

Manual recruiting doesn't scale. ATS systems do keyword matching — no understanding of the JD, no reasoning about seniority proximity, no explanation for low scores.

7-Layer Agentic Pipeline
1 Understand JD — extract skills, seniority, location
2 Generate Boolean queries per source
3 Fetch: Google CSE · Naukri · People Data Labs

5 Score & rank with weighted formula
6 Generate natural-language reasoning per candidate
7 HITL checkpoints — recruiter stays in control

50% Must-have skills matched
25% Nice-to-have skills
15% Seniority proximity

Job Application Agent

Private Repo
